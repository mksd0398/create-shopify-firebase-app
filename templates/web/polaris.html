Modals

Basic Modal

App Bridge modals are rendered by Shopify outside your iframe for a native feel. Use <ui-modal>.

HTML 
<ui-modal id="my-modal">
  <div style="padding: 16px;">
    <p>Modal content goes here.</p>
  </div>
  <ui-title-bar title="Modal Title">
    <button variant="primary"
      onclick="document.getElementById('my-modal').hide()">
      Done
    </button>
  </ui-title-bar>
</ui-modal>

<!-- Trigger -->
<button onclick="document.getElementById('my-modal').show()">
  Open Modal
</button>
Confirmation Dialog

Use a confirmation modal before destructive actions like deleting records.

HTML 
<ui-modal id="confirm-delete">
  <div style="padding: 16px;">
    <p>Are you sure you want to delete this item?
      This action cannot be undone.</p>
  </div>
  <ui-title-bar title="Confirm Deletion">
    <button onclick="document.getElementById('confirm-delete').hide()">
      Cancel
    </button>
    <button variant="primary" tone="critical"
      onclick="handleDelete()">
      Delete
    </button>
  </ui-title-bar>
</ui-modal>

Toasts

Toast Notifications

Toasts provide brief, non-blocking feedback. In embedded admin, they use App Bridge. Outside, a CSS fallback renders.

JavaScript 
// Success toast
showToast("Product saved successfully");

// Error toast
showToast("Failed to save product", true);

// App Bridge native API (also works)
shopify.toast.show("Message", { duration: 5000 });

Resource Picker

Product Resource Picker

App Bridge provides a native product picker. Only available when running inside the Shopify admin.

JavaScript 
// Open the product resource picker
const selected = await shopify.resourcePicker({
  type: "product",
  // Optional: allow multiple selection
  // multiple: true,
  // Optional: filter
  // filter: { query: "shirt" },
});

if (selected && selected.length > 0) {
  console.log("Selected:", selected);
  selected.forEach(product => {
    console.log(product.title, product.id);
  });
}
API Fetch Pattern

Use the apiFetch() helper for all API calls. It handles session tokens automatically.

JavaScript 
// GET request
const shopData = await apiFetch("/api/shop");

// POST request with body
await apiFetch("/api/settings", {
  method: "POST",
  body: JSON.stringify({
    settings: { greeting: "Hello!" }
  }),
});

// Search with query params
const results = await apiFetch(
  "/api/products/search?q=" + encodeURIComponent(term)
);

// Error handling
try {
  const data = await apiFetch("/api/shop");
  showToast("Success!");
} catch (err) {
  showToast("Error: " + err.message, true);
}
